# go-yapi

go-yapi is a [Go](http://golang.org/) client for [Yapi](https://hellosean1025.github.io/yapi/index.html) .

2022.05.08更新
修改api返回数据解析错误的问题，重写api请求调用
## Installation
Install go-yapi using the "go get" command:

```bash
go get github.com/micrease/go-yapi
```
